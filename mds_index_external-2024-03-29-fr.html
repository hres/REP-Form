Processus d’inscription réglementaire (PIR) pour les instruments médicaux

Liens d’accès au modèle et au formulaire du PIR

Processus de transaction réglementaire externe

Modèle de l’entreprise (29 mars 2024)
  • Obligatoire afin d’obtenir un identifiant d'entreprise et/ou fournir des informations à jour sur l'entreprise à Santé Canada.
  • Envoyé par courriel.
    • Joignez l’ébauche de fichier CO XML avant l’envoi.
  • Reportez-vous document d’orientation du PIR pour les instruments médicaux pour une description détaillée des exigences du modèle de l'entreprise.
Formulaire de demande pour les ID de dossiers pour les instruments médicaux
  • Nécessaire pour obtenir un identifiant de dossier.
  • Une fois rempli, le formulaire est envoyé automatiquement par courriel à Santé Canada.

Dans le cadre du processus de transaction réglementaire

Modèle de transaction réglementaire (à venir)
  • Envoyé par la PCDE dans le cadre de la transaction réglementaire (demande).
  • Requis avec chaque transaction réglementaire déposée auprès de Santé Canada.
  • Référez-vous au document d’orientation du PIR pour les instruments médicaux pour une description détaillée des exigences du modèle TR.
Modèle d’information sur les demandes d’homologation (à venir)
  • Envoyé par le PCDE dans le cadre de la transaction réglementaire (demande).
  • Obligatoire uniquement pour un sous-ensemble de transactions. Reportez-vous à la section « 2.5.2 Modèle et processus d'information sur la demande » de l'ébauche du document d'orientation PIR pour les dispositifs médicaux décrivant en détail les exigences relatives au modèle d'information sur les demandes d’homologation.
Détails de l’instrument (29 mars 2024)
  • Envoyé par le PCDE dans le cadre de la transaction réglementaire (demande).
  • Obligatoire pour toutes les nouvelles demandes d’homologation, ainsi que pour toute demande de modification susceptible d’avoir des répercussions sur le répertoire des fabricants.
  • Télécharger détails de l’instrument xlsx fichier
Formulaire d'attestation de demande (29 mars 2024)
  • Envoyé par la PCDE dans le cadre de la transaction réglementaire (demande).
  • Obligatoire pour la première transaction d’une nouvelle activité réglementaire.
  • Télécharger formulaire pdf d'attestation de demande
