Déclaration d'accessibilité pour Processus d'inscription réglementaire (PIR) pour les instruments médicaux

Santé Canada et Agence de la santé publique du Canada s'engagent à rendre leurs sites Web et leurs applications mobiles accessibles, conformément au Règlement canadien sur l'accessibilité.

Statut de conformité

Le site Web a été testé conformément à la norme WCAG 2.2 AA (Règles pour l’accessibilité des contenus Web) et à la norme d'accessibilité EN 301 549 adoptée par le Canada.

Ce site web est entièrement conforme(s) aux Règles pour l’accessibilité des contenus Web (WCAG) version 2.2 niveau AA.

Préparation de cette déclaration d'accessibilité

Cette déclaration a été rédigée le 2026-01-28.

Ce site web a été testé pour la dernière fois le 2026-01-21 selon la norme WCAG 2.2 AA.

Le test a été réalisé par l’équipe accessibilité de Santé Canada.

Commentaires et coordonnées

Pour faire part de vos commentaires ou signaler un obstacle, vous pouvez utiliser la formulaire de commentaires sur l'accessibilité de Santé Canada ou appelez le 1-833-725-2751.
